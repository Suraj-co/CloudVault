# ☁️ CloudVault - Secure Personal Storage System with AI Integration

CloudVault is a modern cloud-based personal storage platform that allows users to securely upload, manage, organize, and access their files from anywhere. The system is enhanced with AI-powered features for intelligent file search, categorization, and storage management.

## 🚀 Features

### 🔐 Authentication & Security
- User Registration & Login
- JWT-Based Authentication
- Role-Based Access Control (RBAC)
- Password Encryption using BCrypt
- Secure API Endpoints

### 📂 File Management
- Upload Files
- Download Files
- Delete Files
- Rename Files
- Create Folders
- Organize Files Efficiently

### 🤖 AI-Powered Features
- Smart File Categorization
- AI-Based Search Recommendations
- Duplicate File Detection
- Storage Usage Insights
- Intelligent File Tagging

### 📊 Dashboard
- Total Storage Used
- Recent Uploads
- File Statistics
- Storage Analytics

### ☁️ Cloud Storage
- Scalable Storage Architecture
- Efficient Metadata Management
- Secure File Access
